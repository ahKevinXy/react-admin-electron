
## 安装

Clone the repo and install dependencies:

```bash
git clone --depth 1 --branch master git@github.com:ahKevinXy/react-admin-electron.git your-project-name
cd your-project-name
npm install
```

## 开发环境

开始调试环境

```bash
npm start
```

## 打包平台

To package apps for the local platform:

```bash
npm run package
```
